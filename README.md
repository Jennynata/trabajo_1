# Simulador de Deshacer/Rehacer (Undo/Redo) - Java 🚀

Proyecto desarrollado para la asignatura de **Ingeniería de Software y Bases de Datos** en la **IUDigital de Antioquia**. El objetivo es comprender y aplicar la estructura de datos **Pila (Stack)** mediante un simulador de editor de texto.

## 📋 Objetivo
Implementar un sistema de gestión de estados utilizando dos pilas manuales:
1.  **Pila Principal:** Almacena las acciones o líneas de texto ingresadas.
2.  **Pila Secundaria:** Almacena temporalmente las acciones deshechas para permitir la función de "Rehacer".



## 🛠️ Requisitos e Instalación
* **Lenguaje:** Java 17 o superior.
* **IDE recomendado:** IntelliJ IDEA, VS Code o Eclipse.
* **Control de Versiones:** Git & GitHub.

### Ejecución
1. Clona el repositorio:
   ```bash
   git clone [https://github.com/Jennynata/trabajo_1.git](https://github.com/Jennynata/trabajo_1.git)

2. Compila el proyecto:

    ```bash
    javac Main.java EditorTexto.java
    Ejecuta la aplicación:

    ```bash
    java Main

3. 🕹️ Funcionalidades
Escribir: Agrega una nueva línea de texto al editor.

Deshacer (Undo): Remueve la última línea y la mueve al historial de rehacer.

Rehacer (Redo): Recupera la última línea deshecha.

Mostrar: Visualiza el estado actual del puntero en la pila.
